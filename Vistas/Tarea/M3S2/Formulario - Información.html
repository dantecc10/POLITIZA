Formulario

Por Dante Castelán Carpinteyro.

¿Qué es un formulario web?

Los formularios web son uno de los principales puntos de interacción entre un usuario y un sitio web o aplicación. Los formularios permiten a los usuarios la introducción de datos, que generalmente se envían a un servidor web para su procesamiento y almacenamiento (consulta Enviar los datos de un formulario más adelante en el módulo), o se usan en el lado del cliente para provocar de alguna manera una actualización inmediata de la interfaz (por ejemplo, se añade otro elemento a una lista, o se muestra u oculta una función de interfaz de usuario).

El HTML de un formulario web está compuesto por uno o más controles de formulario (a veces llamados widgets), además de algunos elementos adicionales que ayudan a estructurar el formulario general; a menudo se los conoce como formularios HTML. Los controles pueden ser campos de texto de una o varias líneas, cajas desplegables, botones, casillas de verificación o botones de opción, y se crean principalmente con el elemento input, aunque hay algunos otros elementos que también hay que conocer. Los controles de formulario también se pueden programar para forzar la introducción de formatos o valores específicos (validación de formulario), y se combinan con etiquetas de texto que describen su propósito para los usuarios con y sin discapacidad visual.

Controles de texto

Campos de texto de una sola línea

Los campos de texto de una sola línea se limitan a permitir la entrada de texto en una "dimensión", que puede ser manipulable en el eje x. La sintaxis es la siguiente:

<input type="text">

Campos de texto multilínea

Los campos de texto multilínea pueden ser ampliados o minimizados debido a un control en su esquina inferior izquierda. La sintaxis es la siguiente:

<textarea></textarea>

Controles de opción

Casillas de verificación

Permiten obtener un valor booleano de su estado para tomar decisiones o determinar una preferencia biopcional:

<input type="checkbox">

Botones de opción

Son para elegir entre una o varias opciones, de forma mutuamente excluyente generalmente, aunque esto no es una regla.

<input type="radio">

Listas

Para elegir fácilmente de entre una lista de opciones (redundantemente).

<select name="genero"> <option>Masculino<option> <option>Femenino</option> </select>

Botón de envío

Recopila la información del formulario para su envío o procesamiento.

<input type="submit">

Botón de borrar o limpiar

Sin envío o procesamiento, y con programación nativa de HTML por defecto, limpia los campos que están en el formulario.

<input type="reset">
